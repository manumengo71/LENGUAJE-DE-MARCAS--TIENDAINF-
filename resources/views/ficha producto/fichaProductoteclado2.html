HP 100

El teclado HP 100 tiene todo lo que necesitas, pero no se queda ahí. Presenta una gama completa de teclas, un perfil cómodo y una configuración sencilla. Conéctalo y empieza a escribir con esta herramienta diseñada para satisfacer tus necesidades.

Caracteristicas

  • Incorpora una completa gama de 107 teclas, incluidas 12 teclas de función y 3 teclas de acceso rápido.
  • La conexión es muy sencilla gracias a la conectividad USB para que puedas usarlo inmediatamente.

Especificaciones

  • Uso recomendado: Oficina
  • Sistema operativo Windows soportado: Windows 10,Windows 7,Windows 8
  • Dimensiones de teclado (Ancho x Profundidad x Altura): 450 x 147,5 x 25,3 mm
